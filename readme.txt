la lista de pokemons va a entregar los nombres que seran un link hacia la info completa de este pokemon
que se consigue en este enponit https://pokeapi.co/api/v2/pokemon/{namePokemon}

La info completa de cada pokemon contiene: Name, Moves(que estos tienen su titulo y descripcion),  tambien tienen su img en sprites(frontdefault)

POKEMON INTERFACE
The pokemon interface contain 3 mains components, those show different data (pokemons, abilities and locations).
Components Main of Pokemon interface: PokemonsList, Abilities and Locations.
The PokemonsList contain a component with list of pokemons and other that show one pokemon.
